# TocaMais - Context

## Architecture
- React SPA (Vite) + Supabase (auth, DB, edge functions)
- Tailwind CSS with custom neon theme (purple/green)
- Framer Motion for animations
- Lucide React for icons

## Key Files
- `src/pages/artist/ArtistTip.jsx` — Public tip/gorjeta page for fans to send music requests + tips via PIX
- `src/pages/artist/ArtistProfile.jsx` — Artist profile management (includes Asaas wallet connection)
- `src/pages/venue/VenueArtists.jsx` — Venue artist listing
- `src/lib/AuthContext.jsx` — Auth context
- `src/lib/supabaseClient.js` — Supabase client
- `src/components/shared/AppLayout.jsx` — Main app layout with header, sidebar, "Ser Pro" button + subscription modal
- `supabase/functions/asaas-create-pix/index.ts` — Creates PIX payment via Asaas API `/v3/payments` with `billingType: 'PIX'` → QR code with amount embedded + copia-e-cola. Also handles `mode: 'subscription'` for Pro
- `supabase/functions/asaas-check-payment/index.ts` — Payment status polling via Asaas API (legacy, no longer used for tips)
- `supabase/functions/asaas-webhook/index.ts` — Webhook receiver: handles PIX_RECEIVED (tips) + subscription lifecycle (revokes is_pro)
- `supabase/functions/asaas-process-tip/index.ts` — Process tip payment, transfer to artist, insert music_request (legacy, now handled by webhook)
- `supabase/functions/asaas-create-subaccount/index.ts` — Creates Asaas subaccount for artist (marketplace flow). Returns walletId + apiKey. Artist receives activation email from Asaas.
- `scripts/migration-pending-tips-and-platform-pix.sql` — Creates `platform_pix_keys` + `pending_tips` tables
- `scripts/migration-add-asaas-wallet.sql` — Migration to add asaas_wallet_id + pix_key columns
- `scripts/migration-add-subscription-id.sql` — Migration to add asaas_subscription_id column
- `scripts/migration-add-subaccount-columns.sql` — Adds asaas_api_key + asaas_account_status columns
