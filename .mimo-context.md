# TocaMais - Context

## Architecture
- React SPA (Vite) + Supabase (auth, DB, edge functions)
- Tailwind CSS with custom neon theme (purple/green)
- Framer Motion for animations
- Lucide React for icons

## Key Files
- `src/pages/artist/ArtistTip.jsx` — Public tip/gorjeta page for fans to send music requests + tips via PIX
- `src/pages/artist/ArtistProfile.jsx` — Artist profile management (includes Asaas wallet connection)
- `src/pages/venue/VenueArtists.jsx` — Venue artist listing
- `src/lib/AuthContext.jsx` — Auth context
- `src/lib/supabaseClient.js` — Supabase client
- `src/components/shared/AppLayout.jsx` — Main app layout with header, sidebar, "Ser Pro" button + subscription modal
- `supabase/functions/asaas-create-pix/index.ts` — PIX payment creation via Asaas API with split
- `supabase/functions/asaas-check-payment/index.ts` — Payment status polling via Asaas API
- `supabase/functions/asaas-create-subscription/index.ts` — Monthly subscription creation (R$ 49,90/mês) via Asaas API
- `scripts/migration-add-asaas-wallet.sql` — Migration to add asaas_wallet_id + pix_key columns
